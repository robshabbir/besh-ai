# Calva Website Audit — Feb 20, 2026

## Issues Found

### Critical (Blocking)
1. **Voice demo widget doesn't auto-start inline** — clicking "Start Conversation" tries to trigger ElevenLabs shadow DOM button which may not be ready; the widget renders hidden off-screen
2. **ElevenLabs widget is hidden** — positioned at bottom: -9999px for "programmatic triggering" but this doesn't actually embed it inline
3. **Test suite broken** — points to port 3000, server runs on 3100
4. **No real-time transcript** in the demo panel — the transcript area is empty, shows only an empty state

### High (User-visible)
5. **Industry selector in demo doesn't change the actual agent** — clicking Law/Medical/etc only changes the label, not the ElevenLabs agent (we only have 1 agent: plumbing)
6. **Old industry-tab panels were removed** — replaced with 3 small cards, losing the rich conversation examples
7. **Hero phone demo plays scripted animations** — not real AI interaction, could confuse users
8. **Voice quality gap** — We use `eleven_flash_v2` which is fast but less natural than `eleven_v3` or competitors

### Medium
9. **Landing page is very long** — 1600+ lines, hard to maintain
10. **No error handling** if mic permission denied
11. **CSP headers** may block ElevenLabs WebSocket connections
12. **Mobile responsive** — two-panel demo layout may break on small screens
